# HOME GO

家政婦・ベビーシッターなど生活サポートスタッフを探せる
マッチングアプリを想定して開発したモバイルアプリです。

---

## 概要

HOME GO は、家事代行スタッフやベビーシッターなど
生活サポートサービスを必要としている利用者が、
条件に合ったスタッフを探せることを目的としたアプリです。

仕事内容・得意分野・料金・対応エリアなどを
分かりやすく表示し、ユーザーが直感的に
サポート人材を見つけられるUIを意識して制作しました。

---

## 主な機能

* サポートスタッフ一覧表示
* スタッフ詳細プロフィール表示
* 条件に応じた検索（想定）
* お気に入り保存機能
* カード型UIによる分かりやすい画面構成
